Getting the IPv4, IPv6 from the User via discord webhook

# Requirement
https://nodejs.org/en/

# Install
1. `npm i cpu-features`
2. `npm i discord-webhook-node`
3. `npm i ip`

# Start
Open your command line and type `node index.js`
